Pre entrega 2 - Nanzer

Descripción del proyecto

Este simulador es una herramienta que permite gestionar las ventas diarias de un pequeño negocio.
La idea de la herramienta en este punto es que permita iniciar el dia de trabajo registrando el stock de cada
tipo de producto de la tienda, agregar o quitar un producto nuevo si fuera necesario, y registrar los pedidos de
día para luego tener estadísticas al final de la jornada.
Las funciones serían:

  1. Crear un nuevo pedido.
  2. Agregar / Quitar productos.
  3. Modificar datos de un producto.
  4. Mostrar estadísticas de la jornada.

Al ingresar se muestra un menu donde se elije si se desea crear un pedido, ver los precios o modificar los precios.

  • Si se desea armar un nuevo pedido, el sistema pide ingresar la cantidad de cada producto. Ingresadas las 5 cantidades, se pregunta el descuento a realizar en porcentaje. Al finalizar, el sistema muestra el precio total del pedido y al dar a aceptar se vuelve al menu inicial.
  • Si se desea ver los precios, el sistema muestra el listado de precios actuales. Al aceptar se regresa al menu principal.
  • Si se desea modificar el precio de los productos, el sistema pide ingresar el valor de cada uno de los productos. Al finalizar se vuelve al menu inicial.
