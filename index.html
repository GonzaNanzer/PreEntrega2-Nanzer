Descripción del proyecto

El proyecto consiste en un sistema de gestion de pedidos. Permite crear pedidos, agregar productos a los ofrecidos por la tienda, modificar datos de los productos que la tienda ofrece como precio y stock, y ver estadísticas de la tienda en la sesión activa.

Funciones:

  1. Crear un nuevo pedido.
  2. Agregar productos.
  3. Modificar datos de un producto.
  4. Mostrar estadísticas:
    • Pedidos del día.
    • Recaudación total.
    • Productos que deben reponerse por tener stock menor a 5 unidades.

Detalles del funcionamiento:

Al ingresar se muestra un menu con las opciones: Crear pedido, Agregar producto, Modificar producto o Ver estadísticas. Si se ingresa un valor erroneo o se presiona cancelar, se muestra un mensaje de alerta y se regresa al menu.

  • Si se desea armar un nuevo pedido: Se pide un nombre de cliente, y luego se muestran los productos disponibles y se solicita escribir los que se desean agregar al pedido. El nombre puede ser cualquier dato. El sistema corrobora que se ingrese correctamente el nombre del producto. Una vez seleccionado el producto, el sistema muestra el stock de ese producto y solicita la cantidad a agregar, controlando la validez del dato. Aceptar sigue cargando productos, y al cancelar se muestra el resumen del pedido.
  • Si se selecciona agregar productos, se solicita nombre (puede ser cualquier dato). Si el producto ya existe, se alerta y se regresa al menu inicial. Luego se pide precio y stock (estos sí se validan). Al aceptar se puede cargar más productos, al cancelar se vuelve al menu.
  • Si se selecciona modificar producto, el sistema muestra los disponibles y pide el nombre del producto a modificar. Si el dato es erroneo, se alerta y se regresa al menu inicial. Si es correcto, se piden y validan los valores de precio y stock. Si se cancela, no se modifican los datos.
  • Si se selecciona ver estadísticas, se muestran con alert sucesivamente: Pedidos del día que no estén vacíos, Recaudación total y nombres de productos con stock menor a 5 unidaes.
